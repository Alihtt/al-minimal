# Favicon Files

Place your favicon files in this directory.

## Required Files

- `favicon.ico` - Main favicon file
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `apple-touch-icon.png` - 180x180 PNG for Apple devices
- `android-chrome-192x192.png` - 192x192 PNG for Android
- `android-chrome-512x512.png` - 512x512 PNG for Android

## How to Generate

Use one of these free favicon generators:

- [favicon.io](https://favicon.io/) - Generate from text, image, or emoji
- [realfavicongenerator.net](https://realfavicongenerator.net/) - Comprehensive favicon generator

## Steps

1. Create or choose an image (square, at least 512x512px)
2. Upload to a favicon generator
3. Download the generated files
4. Place all files in this directory
5. Update `site.webmanifest` in the root directory with your blog name

That's it! The template already has the favicon links configured in `_layouts/default.html`.
