# Features

Complete guide to all blog features.

## Blog Features

### Pagination

- 10 posts per page (configurable)
- URLs: `/articles/`, `/articles/page/2/`, etc.
- Auto-generated page numbers
- Previous/Next navigation

**Configure:** Edit `per_page` in `_config.yml`

### Tags System

- Organize posts by tags
- Auto-generated tags page at `/tags`
- Click any tag to see all posts
- SEO-friendly tag pages

**Usage:** Add `tags: [python, tutorial]` to post front matter

### Table of Contents

- Auto-generated TOC for long posts
- Smooth scrolling to sections
- Active section highlighting
- Mobile-friendly

**Usage:** Add `toc` array in post front matter (see [CONTENT.md](CONTENT.md))

### Social Sharing

Share buttons on every post and project:
- Twitter/X
- LinkedIn
- Facebook
- Reddit
- Telegram
- Copy Link

Buttons auto-fill title and URL.

### RSS Feed

- Auto-generated at `/feed.xml`
- Include in your posts
- SEO-friendly

Readers can subscribe via feed readers.

## SEO Features

### Meta Tags

- Title tags
- Description meta tags
- OpenGraph tags (social sharing)
- Twitter Card tags
- Canonical URLs

Auto-generated by `jekyll-seo-tag` plugin.

### Sitemap

- Auto-generated at `/sitemap.xml`
- Submit to Google Search Console
- Helps search engines index your site

### Robots.txt

- Controls search engine crawling
- Located at `/robots.txt`
- Links to your sitemap

### Clean URLs

- No `.html` extensions
- SEO-friendly structure
- Example: `/articles/my-post/`

## Developer Features

### GitHub Actions

- Auto-deployment
- Build on push
- No manual steps
- Error notifications

### Live Reload

Development mode:
```bash
bundle exec jekyll serve --livereload
```

Changes reload automatically.

### CSS Hot Reload

Watch mode:
```bash
npm run watch:css
```

Tailwind rebuilds on changes.
